# BELSI.Driver

Android-приложение для водителей — доставка строительных материалов.

## Стек

- Kotlin + Jetpack Compose
- Material Design 3
- Hilt DI
- Room Database
- Retrofit + OkHttp
- CameraX
- WorkManager

## Сборка

1. Клонировать репозиторий
2. Создать `local.properties` с `sdk.dir=/path/to/Android/sdk`
3. `./gradlew assembleDebug`
